生命是一條多變的旅程，人類社會又是一個多變的複合體，兩者之間如何互動？如何融合？都是成長過程中最大的探索領域。

在四天完全迥異的生活環境中，學員將學習到:

  • 面對不同場合的態度和禮儀及適應能力
  • 在陌生環境中破解難題的自信
  • 貧乏與富裕的比較和接受
  • 對周遭世界的諒解

第一日第一夜 {野外生活 }
學員須在導師帶領下入住自己搭建的營幕，
並解決野外生活各種所需。

達致目標與期望自律

  • 自信心與自我照顧能力
  • 面對貧乏生活條件的生存技巧

學習範疇

  • 行軍式遠足
  • 搭建營幕
  • 野外煮食
  • 以大自然素材製作生活用品
  • 個人生活所有操作

[第二日第二夜 { 鄉村生活 } ]
學員將由導師陪同入住僻遠鄉郊的傳統客家村屋或漁村，
與原居民共度兩日一夜。

達致目標與期望

  • 與人相處之道
  • 勤奮工作的回報
  • 面對簡樸的生活條件
  • 對不同社群的尊重

學習範疇

  • 鄉村生活與人際關係模式
  • 禮儀與禁忌
  • 鄉郊生活操作，包括:
    • 清理
    • 修補工程如籬笆 、雞籠 、油漆等
    • 採集食物
    • 照顧農作物及飼畜

[第三日第三夜 {城市生活體驗 } ]
學員將入住設備齊全的豪華酒店，享用各種服務及活動，並將出席音樂會、話劇或藝術展覽等文化活動。

達致目標與期望

  • 與不同階層或國籍的陌生人相處
  • 城市生活應有禮儀
  • 城市生活應有技巧

學習範疇

  • 出席文化活動
  • 與外國遊客溝通及建立友誼
  • 如何要求及與服務人員溝通
  • 西式餐桌禮儀
  • 享用服務時應有的禮貌
  • 作為策劃者安排小組城市活動

[第四日{總結,交流與分享 } ]
在導師的帶領下，學員將與同學分享三日來的經驗，並作一簡短的演講。

達致目標與期望

  • 表達技巧與及說服力
  • 瞭解與接受他人的感受及想法

學習範疇

  • 總結經驗
  • 表達自己的感受及想法
